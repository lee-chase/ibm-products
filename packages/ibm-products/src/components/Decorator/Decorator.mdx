## Overview

The Decorator groups a key/value pair to look and behave like a single UI
element.

## Decorator as Link code sample

```jsx
<Decorator
  label="IP"
  href="http://www.ibm.com"
  kind="link"
  onClick={(event, values) => {
    /* Your code here. */
  }}
  onContextMenu={(event, values) => {
    /* Your code here. */
  }}
  score={5}
  value="192.168.0.50"
/>
```

By giving the `href` prop a value, this Decorator will be rendered as an anchor
(`<a>`). In this case, other anchor-related properties can be included, such as
adding `target="_blank"` to open in a new browser tab.

```jsx
<Decorator
  label="IP"
  href="http://www.ibm.com"
  kind="link"
  target="_blank"
  score={5}
  value="192.168.0.50"
/>
```
